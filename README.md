## 🚀 Características

- **🔍 Búsqueda Híbrida**: Full-text + semántica con embeddings
- **🔗 Conexiones Inteligentes**: Detección automática de relaciones
- **🤖 MCP Server**: Integración para agentes de IA (Cursor, Claude Code)
- **🔄 Sincronización**: Webhooks de GitHub automáticos
- **⚙️ CLI Tool**: Administración desde línea de comandos

---

## �️ Stack

- **Backend**: Elixir/Phoenix
- **Base de Datos**: PostgreSQL + Qdrant (búsqueda vectorial)
- **Deployment**: Docker Compose

## 🛠️ Quick Start

```bash
# 1. Clonar y configurar
git clone https://github.com/nucleoabierto/docs-alejandria.git
cd alejandria
cp .env.example .env

# 2. Iniciar servicios
docker-compose up -d postgres qdrant

# 3. Instalar dependencias y migrar
mix deps.get && mix ecto.migrate

# 4. Iniciar servidor
mix phx.server
```

**Guía completa:** [Development Setup](./.context/engineering/guide/development-setup.md)
